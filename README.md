# 🏪 Store System  
### 🚀 Inventory • Orders • Customers • Payments • Smart Backup

> A full-stack store management system built with **React + Node.js + SQLite**, designed for simplicity, performance, and real-world usability.

---

## ✨ Overview

Store System is a powerful yet lightweight application that helps you manage:

- 📦 Products & Inventory  
- 👥 Customers  
- 🧾 Orders & Billing  
- 💰 Payments & Due Tracking  
- 📊 Stock Movement History  
- 🔄 Automatic Database Backups  

Built with a focus on **clean architecture**, **local performance**, and **real usability**.

---

## 🧠 Key Features

| Feature | Description |
|--------|------------|
| 📦 Product Management | Add, update, deactivate products |
| 👥 Customer System | Manage customer details & history |
| 🧾 Order System | Create & track orders |
| 💰 Payments | Track paid & remaining amounts |
| 📊 Stock Tracking | Automatic stock updates with history |
| 🔄 Auto Backup | Daily database backup (cron-based) |
| ⚡ SQLite | Fast, serverless database |
| 🧩 Modular Backend | Clean routes, controllers & utils |

---

## 🛠️ Tech Stack

### 🔷 Frontend
- React (Vite)
- JavaScript
- CSS

### 🔶 Backend
- Node.js
- Express.js
- SQLite3

### ⚙️ Tools & Utilities
- node-cron → scheduled backups  
- dotenv → environment variables  
- concurrently → run frontend + backend
